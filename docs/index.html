Kiln Cocos Creator 2 Package

Github Repository

Supported Features:
  • Leaderboards
  • In App Purchases
  • Analytics
  • Rewarded Ads
  • Interstitial Ads
  • Banner Ads
  • Android Export

Integration:

To integrate this package into your Cocos Creator 2 Projects, clone the repository or download a release from Releases and put the kiln folder in ~/.CocosCreator/packages (Windows users should use the following path instead: C:\Users\${your-name}\.CocosCreator\packages). Alternatively you can put the folder in ${your-project}/packages instead which will make the package available only on ${your-project} instead of globally.

Once you've integrated the package you might need to re open the project if it was already loaded, and once you do that you'll find a new Menu

First step is to "Initialize Extension", which will copy some resources into your project's asset folder.

At that point all that's left to do is open the Settings panel through the Menu and configure your project's properties.

This package provides tools for mocking behavior in Editor, so you can configure if you wish for your In Editor Platform to support different features, and as well configure placements for Ads, In App Purchases, Leaderboards and Analytic Events.

Included with this package is a Demo Application that you can load and explore, to see how the different features supported through Kiln's API can be used

API Examples

Initialize

Before making use of anything else, it is necessary to initialize the SDK

try {
    await Kiln.API.init();

    cc.log("Kiln Initialized");
}
catch (ex) {
    cc.error("Error Initializing Kiln");
    cc.error((ex as Error).message);
}

Load and Display Rewarded Ad

 
try {
    await Kiln.API.loadRewardedAd(placementId);

    cc.log(`Rewarded Ad '${placementId}' displayed. Should reward: ${result.withReward}`);
}

Load, Display, Hide and Destroy Banner Ads

 
try {
    // Select a Placement Id
    placementId = await this._idSelector.selectID(Kiln.EditorSettings.banners);
    
    // Banner Alignment and Size
    const position = KilnBannerPosition.BottomCenter;
    const maxSize = KilnBannerSize.Width320Height50;

    await Kiln.API.loadBannerAd(placementId, position, maxSize);

    cc.log(`Banner Ad '${placementId}' Loaded`);

    await Kiln.API.showBannerAd(placementId);

    cc.log(`Banner Ad '${placementId}' displayed`);

    // Banners can be hidden as well
    await Kiln.API.hideBannerAd(placementId);

    cc.log(`Banner Ad '${placementId}' hidden`);

    // And when you're done with them, they can be destroyed
    await Kiln.API.destroyBannerAd(placementId);

    cc.log(`Banner Ad '${placementId}' destroyed`);
}
catch (ex) {
    cc.error((ex as Error).message);
}
